# Kouvee Pet Shop Desktop Application

![Preview](screenshot/Screenshot.png)
This is the desktop version of Kouvee Petshop for P3L's final submission.
## Developer Guide
### Requirement
1. Open JDK 13 or newer
2. IntelliJ IDEA
3. Scene Builder
4. JavaFX SDK 12 or newer
### IDE Plugin
1. Lombok
